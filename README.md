# Ray Tracing


## Introduction

This is the CUDA implementation of ray-tracing algorithm. The project was initially inspired by [Ray Tracing In One Weekend/The Next Week](https://github.com/RayTracing/raytracing.github.io)


## How to run

```
cd build
cmake ../
make
./Raytracing
```
